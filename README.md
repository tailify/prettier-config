> Tailify's very own opinionated Prettier configuration.

## Installation

```
$ npm install --save-dev @tailify/prettier-config
```

## Usage

To get started, add this to your `.prettierrc.js` file:

```js
module.exports = require('@tailify/prettier-config');
```
